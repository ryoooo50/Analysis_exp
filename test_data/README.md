# ピーク分析テストデータ

## peak_test_data.csv

### データ仕様
- サンプリング周期: 0.01秒 (100Hz)
- 角度範囲: 0° → 50° → 0° の往復運動
- サイクル: 3回繰り返し (0.2秒周期)
- 関節データ: Lower_Arm_R/rotation_y のみ

### 期待される結果
- **角速度（フィルタ前）**: 約 500 deg/s = **8.73 rad/s**
  - 計算: (50° - 0°) / 0.1s = 500 deg/s
  - rad/s変換: 500 * (π/180) ≈ 8.73 rad/s
- **ピーク数**: ローパスフィルタ後のパラメータ設定による
